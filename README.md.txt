# E-Commerce Database SQL Project

This project demonstrates SQL database design and business reporting using an e-commerce system.

# Features
- Database creation
- Table relationships
- Data insertion
- SQL JOIN operations
- Revenue calculations
- Customer order tracking
- Product sales analysis

# Tables
- Customers
- Products
- Orders
- OrderDetails
- Payments

# Reports
- Customer order history
- Best-selling products
- Revenue by product
- Top customers
- Total sales revenue

# Tools Used
- SQL Server
- SQL Server Management Studio (SSMS)
